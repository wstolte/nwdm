5 More on WFS

Full documentation is found here: https://docs.geoserver.org/latest/en/user/services/wfs/reference.html

The capabilities of the used geoserver schema can be requested by: https://marineprojects.openearth.nl/geoserver/NWDM/ows?service=WFS&version=1.0.0&request=GetCapabilities

A description of the featuretypes (layers) can be requested by: https://marineprojects.openearth.nl/geoserver/NWDM/wfs?request=DescribeFeatureType
